Soy Ingeniero en Comunicaciones y Electrónica, desempeñándome en la industria de la Televisión broadcast, especialmente en el soporte técnico para la operación del Control Maestro y procesamiento de la señal para su difusión en ATSC.

Muchas de las aplicaciones que empleo corren en servidores con Windows Server y sistemas Linux.

Me apoyo en la programación en Delphi para automatizar el manejo de archivos para las guías de transmisión, formateo de los archivos para metadatos del video y la carga en las bases de datos de los los programas que llegan del área de edición.

Manejo SQL para la administración de las bases de datos de los materiales para la transmisión y la administración de los receptores satelitales que se autorizan para recibir la señal encriptada.

En mi interés por las inversiones he analizado HTML para la obtención de datos históricos como la inflación, tipos de cambio y acciones de empresas; y con herramientas de Frontend realizo algunas páginas web para la presentación y consulta de los datos.

Actualmente estoy ampliando mi perfil con Python, github y estoy tomando cursos de Data Scence e Inteligencia Artificial.
